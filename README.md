## 🎯 Overview

**CareerChoice360** is an integrated digital platform designed to bridge the gap between academic education and real-world corporate opportunities. This responsive landing page serves as the primary entry point for prospective students, providing comprehensive insights into the campus infrastructure, academic programs, alumni achievements, and admission procedures.

The platform streamlines the admission enquiry process while highlighting the institution's legacy of excellence, robust placement records, and industry partnerships.

---

## ✨ Key Features

- **Responsive Architecture**: Fluid UI/UX design ensuring seamless accessibility across desktop, tablet, and mobile devices.
- **Dynamic Program Exploration**: Interactive, tab-based navigation for exploring various undergraduate and postgraduate disciplines (Engineering, Law, Management, Commerce, and Education).
- **Integrated Admissions Workflow**: A streamlined, client-side validated enquiry form to facilitate instant communication with the admissions team.
- **Data-Driven Highlights**: Engaging statistical displays showcasing institutional metrics such as placement rates, highest packages, and alumni network size.
- **Alumni Success Stories**: Curated testimonials providing authentic perspectives from successful graduates.

---

## 🛠 Technology Stack

This project is built utilizing a lightweight, dependency-free frontend stack to ensure high performance and broad browser compatibility:

- **Markup**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom styling, Flexbox, CSS Grid)
- **Scripting**: Vanilla JavaScript (ES6+ for DOM manipulation and interactive state management)

## 🚀 Getting Started

### Prerequisites

No external dependencies, build tools, or package managers are required to run this project.

### Installation & Usage

1. **Clone the repository** (or download the source code):
   ```bash
   git clone <repository-url>
   ```
2. **Navigate to the project directory**:
   ```bash
   cd careerchoice360
   ```
3. **Launch the application**:
   Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Safari, Edge).
   *Alternatively, you can use a local development server like VS Code Live Server for a better development experience.*
